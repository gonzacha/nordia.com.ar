# Nordia Landing (v1)

Landing de guerra para vender **Nordia WhatsApp IA**.

## Requisitos
- Node.js 18+

## Ejecutar
```bash
npm install
npm run dev
```

## Configurar WhatsApp
Crear un archivo `.env.local`:

```bash
NEXT_PUBLIC_WA_PHONE=549XXXXXXXXXX
```

## Deploy
Subir a GitHub y desplegar en Vercel.
